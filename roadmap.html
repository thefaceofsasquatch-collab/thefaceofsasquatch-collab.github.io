Priority Upgrades
📅
URL Date Parameters — Filter by Date Range via URL
Allow the Dashboard and Encounter Browser to accept ?from=YYYY-MM-DD&to=YYYY-MM-DD URL parameters. With no parameters, the page shows all data from oldest to newest video. With parameters, every stat, chart, and sighting list is filtered to that exact window. Shared links always reflect the exact date range they were built for — no ambiguity, no stale labels.

Encounter Browser: Low effort — data is already embedded, filter system already exists.
Dashboard: Medium effort — stats need to be migrated from hardcoded numbers to live JS calculations from embedded data.
Feature Medium Effort Planned

Report Archive — Versioned Past Reports at Permanent URLs
Each published report is saved as a dated snapshot (e.g. /reports/2026-april/) and remains accessible forever at its own permanent URL. An archive index page at /reports/ lists all published reports with their date ranges and links. Becomes genuinely valuable once 6–12 months of data exist and month-over-month comparison is meaningful.

Random Sampling Pipeline — Process N Videos Per Month Across All Channels
Expand the channel list and process a configurable number of videos per monthly run (e.g. 40–60) selected randomly from all unprocessed videos across all channels. Prevents runaway API costs while keeping the database growing with broad cross-channel coverage. A single budget number in the config controls monthly spend.
Infrastructure Medium Effort Planned
🔬
Expanded Physical & Sensory Schema — New Data Fields for All Future Videos
Add five new structured fields to the AI extraction prompt and database schema, captured for all videos processed going forward. Existing videos will be backfilled in a future re-processing pass.

Animal Reactions — Dogs reacted / refused area, livestock disturbed, wildlife fled, birds went silent, pets killed or injured, no animal reaction noted.

Physical Size — Estimated height in feet (numeric range) and build descriptor (Massive, Athletic, Lean, Hunched, Unknown).

Eye Description — Color (Red, Amber/Yellow, Green, White/Silver, No eye shine, Not seen) and whether reflective/tapetum shine was noted.

Teeth — Described (large, flat, prominent, fanged, discolored), glimpsed but not described, or not mentioned.

Smell / Odor — Sulfur, rotting flesh, wet dog, musky/animal, skunk, no odor noted, or other described odor. One of the most cross-culturally consistent details in encounter reports.
Infrastructure Medium Effort Planned
📅
Weekly Processing Pipeline — Configurable Budget with Backlog Prioritization
Replace the monthly-only processing model with a weekly run on a configurable video budget (recommended: 30–40/week). Each weekly run allocates videos as follows:

New content — all videos published in the last 7 days across all monitored channels (typically 10–20).

Backlog — 1 per channel — for each channel, rank all unprocessed backlog videos by view count (descending), randomly sample one from the top 20% most-viewed. Balances historical significance with discovery.

Remainder — fills from a random cross-channel backlog pool up to the weekly budget cap.

At 35 videos/week the database grows ~140/month and ~1,700/year. Estimated cost: $5–10/week.
Infrastructure Medium Effort Planned
